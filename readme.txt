# Fishtank Live Report Creator

## Description
A Python-based application to create event reports for the Fishtank Live Discord server.

## Features
- Generate event reports with customizable timestamps and reporter notes.
- Copy reports to the clipboard.
- Windows and macOS versions available.

## Installation and Usage
### Running the App (Pre-packaged Executables)
1. Navigate to the `dist` folder and choose your platform:
   - `dist/mac` for macOS
   - `dist/windows` for Windows
2. Run the application by double-clicking the appropriate file.

### Running from Source
1. Navigate to the `src` folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
3. python fishtank_live_report_creator.py
